# OsReleaseNet.Extensions.DependencyInjection

[![NuGet](https://img.shields.io/nuget/v/OsReleaseNet.Extensions.DependencyInjection.svg)](https://www.nuget.org/packages/OsReleaseNet.Extensions.DependencyInjection/)  [![NuGet](https://img.shields.io/nuget/dt/OsReleaseNet.Extensions.DependencyInjection.svg)](https://www.nuget.org/packages/OsReleaseNet.Extensions.DependencyInjection/)


Enables easy Dependency Injection registration for OsReleaseNet with Microsoft.Extensions.DependencyInjection.

OsReleaseNet provides Linux and FreeBSD OsRelease detection, SteamOS detection, and helpful detection code for Linux.

### Compatibility
OsReleaseNet supports:
* .NET Standard 2.0
* .NET 8
* .NET 9
* .NET 10

## How to build the code
To build the project in Debug mode, run the following command: ``dotnet build -c Debug``.

To build the project in Release mode, run the following command: ``dotnet build -C Release``
